Core SERP Vitals - Settings Page

Show the Core Web Vitals data from the Chrome User Experience Report in the Google Search Results.

Website: https://defaced.dev/tools/core-serp-vitals/

GitHub: https://github.com/workeffortwaste/core-serp-vitals

Setup

This extensions requires a Chrome UX Report API key.

You can obtain a key from this page: https://developers.google.com/web/tools/chrome-user-experience-report/api/guides/getting-started

Once you have a key enter in the box below and click save.
